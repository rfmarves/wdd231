Site Name

The website name is FHE.support

This site name was selected because the site is dedicated supporting families with creative and wholesome Family Home Evening activities.

The domain www.fhe.support is also currently available.

Site Purpose

A website that will help with family home evening each week by providing hymn karaoke, dad jokes, game options, and lesson ideas.

Scenarios

These are the questions the audience could have:

  • What hymn could we sing today?
  • What game could we play today?
  • What fun jokes could we add to our fun time today?
  • What lesson topic should we cover today?
  • Where can we find material for those lesson topics?
  • What videos could we use for a hymn karaoke?
